# DailyChallenge-2026-LeetCode

A curated set of LeetCode solutions and daily practice notes for 2026. Each
solution lives at the repository root and follows the standard LeetCode C++
template, making it easy to open a file and submit directly.

## What you will find
- One file per problem, named as `ID.Title.cpp`.
- Clean, focused C++ solutions intended for LeetCode submissions.
- A lightweight structure with no build system and minimal tooling.

## Why this repo exists
To keep daily problem solving organized, searchable, and consistent across the
year, while preserving the exact code format expected by LeetCode.
